
# GraphQL Practice

This project is an introduction to GraphQL implementation in a NestJS environement.

## Documentation

This project uses 2 tables tables : 
1. Client 
2. Company

###### Routes

<b>Create</b> : Client, Company

<b>Read</b> : Clients, ClientById

<b>Update</b>  : Client

<b>Delete</b>  : Client
